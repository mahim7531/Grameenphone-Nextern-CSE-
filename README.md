# 🧠 SECTION 1: Number Series

### Q1:

2, 3, 5, 9, 17, ?

👉 Pattern: +1, +2, +4, +8
👉 Next = 17 + 16 = **33**

---

### Q2:

1, 1, 2, 6, 24, ?

👉 Pattern: Factorial
👉 1!, 1!, 2!, 3!, 4!
👉 Next = 5! = **120**

---

### Q3:

3, 4, 7, 11, 18, ?

👉 Pattern:
3+4=7
4+7=11
7+11=18
👉 Next = 11+18 = **29**

---

### Q4:

4, 6, 9, 13, 18, ?

👉 Pattern: +2, +3, +4, +5
👉 Next = +6 → **24**
